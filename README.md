# Luxora — Luxury E-commerce UI

React + Vite + Tailwind CSS 4 asosida yasalgan demo onlayn-do'kon sahifasi.

## Ishga tushirish

```bash
npm install
npm run dev
```

Keyin terminalda chiqqan manzilni (masalan `http://localhost:5173`) brauzerda oching.

## Nima bor?

- Navbar + Hero
- Trust bar + Featured Categories
- Flash Sale (jonli sanoq bilan)
- Best Sellers / New Arrivals (Quick View, sevimlilar, savatga qo'shish)
- Popular Brands
- Mijozlar fikri (Testimonials)
- Chegirma banneri + Newsletter
- Footer
- To'liq ishlaydigan savat (cart) tizimi va login modal (demo, real backend'siz)

Hammasi `src/App.jsx` faylida — bitta fayl, boshqa komponent fayllari yo'q.
